# OIBSIP_DataAnalytics_Task1

# 📊 Retail Sales Exploratory Data Analysis

This project performs an in-depth Exploratory Data Analysis (EDA) on a Retail Sales Dataset using Python. The objective is to analyze customer purchasing behavior, product category performance, sales trends, and revenue generation patterns.

The analysis combines statistical techniques and data visualization methods to uncover meaningful business insights that can support decision-making, inventory management, customer targeting, and sales optimization.
---
## Project Concepts and Challenges

* Perform data loading and preprocessing.
* Explore customer demographics and purchasing behavior.
* Analyze product category performance.
* Study sales and revenue patterns.
* Perform descriptive statistical analysis.
* Identify high-revenue products and customer segments.
* Analyze monthly sales trends using time series techniques.
* Generate actionable insights through data visualization.
---
## Dataset Information

The dataset contains retail transaction records including customer details, product information, and sales data.

| Feature          | Description                   |
| ---------------- | ----------------------------- |
| Transaction ID   | Unique transaction identifier |
| Date             | Date of purchase              |
| Customer ID      | Unique customer identifier    |
| Gender           | Customer gender               |
| Age              | Customer age                  |
| Product Category | Product category purchased    |
| Quantity         | Number of products purchased  |
| Price per Unit   | Cost per product              |
| Total Amount     | Total transaction value       |
